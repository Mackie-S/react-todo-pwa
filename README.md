# react-todo-pwa
React Todo PWA project using Firebase Auth &amp; Cloud Firestore

## Getting Start
1. git clone
```
$ git clone 
```
2. npm install
```
$ npm install
```
if you can't install, try below command.
```
$ npm install --legacy-peer-deps
```
3. npm start
```
$ npm start
```

You can use it!!
